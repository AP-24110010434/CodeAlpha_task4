# Disease Prediction from Medical Data

## Objective

Predict the possibility of diseases using patient medical information.

---

## Approach

Machine Learning classification algorithms are used to predict whether a patient is likely to have a disease.

Models Used:

- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest

---

## Dataset

Pima Indians Diabetes Dataset

Features:

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

Target:

- Outcome
  - 0 = No Disease
  - 1 = Disease

---
